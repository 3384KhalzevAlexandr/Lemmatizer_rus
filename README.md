# Лемматизатор русского языка

Программа для лемматизации русского текста с использованием словаря OpenCorpora и частотной модели.

## Структура проекта

- `src/` - исходный код программы
- `models/` - сохраненные модели (словарь, частотная модель, золотой корпус)
- `data/` - исходные XML файлы (не включены в репозиторий)

## Использование

Запуск в корневой директории:

`python -m src.main`
